## 🎯 Objective
To design a **clean, collapsible sidebar** using only CSS.  
The goal was to understand layout structure, hover transitions, and active states — without relying on JavaScript.

---

## 🧩 Features
- Collapsible sidebar navigation  
- Smooth hover effects  
- Icons and labels for each item  
- Minimal and responsive design  

---

## 🧠 What I Learned
- Flexbox alignment for layouts  
- Using transitions for interactivity  
- Reusable class structure in CSS  
- Maintaining visual balance and spacing  

---

## ⚙️ Technologies Used
- HTML5  
- CSS3 (Flexbox, Transitions)
